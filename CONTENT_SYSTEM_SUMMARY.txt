🔧 TECHNICAL DETAILS (FOR DEVELOPERS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Content Loader System:
  - JavaScript module: js/content-loader.js
  - Fetches .txt files from content/ folder
  - Parses [LABEL] format
  - Dynamically updates DOM elements
  - Runs on page load

File Format:
  - Simple key-value pairs
  - [LABEL] = Key
  - Text below = Value
  - Comments start with #
  - Multi-line values supported

Loading Process:
  1. Browser loads index.html
  2. content-loader.js initializes
  3. Fetches all .txt files
  4. Parses content
  5. Updates page elements
  6. Page displays with content
