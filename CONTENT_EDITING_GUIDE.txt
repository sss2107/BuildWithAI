✏️ HOW TO EDIT CONTENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Open the file you want to edit
   - Use any text editor (TextEdit, VS Code, Notepad, etc.)
   - Open the .txt file from the content/ folder

Step 2: Find the section you want to change
   - Look for [SECTION_NAME] in square brackets
   - These are the labels for each piece of content

Step 3: Edit the text BELOW the label
   - Don't change the [LABELS] in square brackets
   - Edit only the text below each label
   - You can change, add, or remove text

Step 4: Save the file
   - Save using Ctrl+S (Windows) or Cmd+S (Mac)

Step 5: Refresh your browser
   - Press F5 or Ctrl+R to see changes
   - Content will update automatically!

📝 FILE FORMAT EXPLAINED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Example:
  # This is a comment (ignored)
  
  [TITLE]
  This is the content that will appear on the website
  
  [SUBTITLE]
  This is another piece of content

RULES:
✓ Lines starting with # are comments (ignored)
✓ [LABELS] in square brackets are section identifiers
✓ DON'T change [LABELS], only change the text below them
✓ You can have multi-line content
✓ Empty lines are okay

💡 QUICK TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ DO:
  • Edit text below [LABELS]
  • Add new lines within sections
  • Save and refresh browser to see changes
  • Keep backups of your content files
  • Use simple text (avoid special characters like <, >)

❌ DON'T:
  • Change [LABELS] in square brackets
  • Delete [LABELS] unless removing entire section
  • Use HTML tags (unless you know what you're doing)
  • Forget to save the file!

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Changes don't appear
Solution:
  1. Make sure you saved the file (Ctrl+S / Cmd+S)
  2. Hard refresh browser (Ctrl+Shift+R / Cmd+Shift+R)
  3. Check browser console for errors (F12)
  4. Make sure [LABELS] are still correct

Problem: Content looks broken
Solution:
  1. Check if you accidentally changed a [LABEL]
  2. Make sure [LABELS] are on their own line
  3. Restore from backup if needed

Problem: Can't find a section
Solution:
  1. Use Ctrl+F (Cmd+F) to search in file
  2. Search for keywords from the website
  3. Check the file structure guide above
