🚀 QUICK GITHUB PAGES DEPLOYMENT COMMANDS
================================================================================================

📋 STEP 1: Navigate to project folder
cd /Users/sahil_sharma/Downloads/BuildWithAI

📋 STEP 2: Initialize Git (if not already done)
git init

📋 STEP 3: Add all files
git add .

📋 STEP 4: Create first commit
git commit -m "Initial commit: Portfolio website"

📋 STEP 5: Rename branch to main
git branch -M main

📋 STEP 6: Add GitHub remote
⚠️ Replace 'yourusername' with your actual GitHub username!
git remote add origin https://github.com/yourusername/BuildWithAI.git

📋 STEP 7: Push to GitHub
git push -u origin main

📋 STEP 8: Enable GitHub Pages
1. Go to: https://github.com/yourusername/BuildWithAI
2. Click "Settings" tab
3. Click "Pages" in left sidebar
4. Under "Branch", select "main" and "/" (root)
5. Click "Save"
6. Wait 1-2 minutes

🎉 DONE! Your site is live at:
https://yourusername.github.io/BuildWithAI/

================================================================================================

🔄 UPDATING YOUR WEBSITE (After Changes)
================================================================================================

After editing any file:

git add .
git commit -m "Update: describe what you changed"
git push origin main

Wait 1-2 minutes, then refresh your live site!
